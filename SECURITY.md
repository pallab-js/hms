# Security Policy

## Supported Versions

Currently only the `main` branch is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Please report security vulnerabilities by opening a private issue on GitHub if possible, or by contacting the project maintainers directly via email.

We will acknowledge your report within 48 hours and provide a timeline for a fix. Please do not report security vulnerabilities through public GitHub issues.
